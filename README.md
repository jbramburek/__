Dunder

## Getting started ##

### Setup environment ###

- Install `python2`
- install virtualenv
  - `pip install virtualenv`
- create virtualenv `dunder`
  - `virtualenv dunder`
  - `source dunder/bin/activate`
- install dependencies
  - `pip install -r requirements.txt`
  
### Run dunder ###

```bash
export SLACK_BOT_TOKEN=<BOT_TOKEN>
./slackbot.py
```
